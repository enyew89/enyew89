# 👨‍💻 About Me

I'm **Enyew**, a third-year **Information Systems** student from **Ethiopia 🇪🇹** who enjoys building software that solves real-world problems.

I'm passionate about understanding **how technologies work behind the scenes**, especially authentication, backend systems, databases, and modern web applications. Rather than simply following tutorials, I enjoy digging deeper into concepts and building projects that strengthen my understanding.

Currently, I'm focused on becoming a well-rounded **Full-Stack Developer**, with a strong interest in backend development, clean architecture, cybersecurity, and Android development.

- 🌱 Currently learning **System Design, Backend Architecture & Advanced Authentication**
- 🔐 Interested in **Web Security & Authentication**
- 💻 Building full-stack applications with **Node.js, Express & MongoDB**

- 🚀 Always looking for opportunities to learn and improve
